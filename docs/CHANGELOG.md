# Changelog

All notable changes to this project will be documented in this file.

## Versions:

### [0.1.2]
- Station-level-aware action resolution: configure your crafting station levels in `station_levels.yaml` and conditional item actions (e.g., "Keep until upgrade is complete; sell once done") are automatically resolved based on your progress
- Added `pyyaml` dependency
- Improved overlay color matching for compound action strings

### [0.1.1]
- Performance improvements
- Code cleanup

### [0.1.0]
- Initial version of the project
- Base OCR tool
- Calibration tool
- Resolution profiles
- Documentation
- Items database + management (via calibration tool)
